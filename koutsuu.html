文法句型

  • 試著用句型造句
# 句型結構 中譯 例句
1 場所 + に + N は + どこですか 在場所裡，Ｎ在哪裡呢？
  • ➀、駅に切符売り場はどこですか。
  • 在車站裡，售票處在哪裡？
  • ➁、コンビニにお菓子はどこですか。
  • 在便利商店，零食在哪裡？
  • ※ 詢問某事物（Ｎ）在某個地點（場所）的位置。「どこですか」表達「在哪裡？」。
2 場所 + まで + N は + どこですか 到場所的Ｎ在哪裡呢？（Ｎ在某地的哪裡？）
  • ➀、駅までトイレはどこですか。
  • 從車站到廁所怎麼走？
  • ➁、駅までコインロッカーはどこですか。
  • 從車站到投幣寄物櫃怎麼走？
  • ※ 「まで」表示目的地或終點，意指「到」。
3 場所 + で + V ます 在場所做Ｖ
  • ➀、駅で電車を待ちます。
  • 在車站等電車。
  • ➁、電車で次の駅を待ちます。
  • 在電車上等下一站。
  • ※ 用來描述「在某地進行某個動作」。
4 Ｎ + まで + の + 切符 + を + ～枚 + ください 請給我到Ｎ的車票～張
  • 京都までの切符を三枚ください。
  • 請給我三張到京都的車票。
  • ※ 「まで」強調到達京都的票，用於描述最終目的地。
5 Ｎ + 行き + の + 切符 + を + ～枚 + ください 請給我前往Ｎ的車票～張
  • 東京行きの切符を四枚ください。
  • 請給我四張往東京的車票。
  • ※ 「行き」用來強調出發的方向。
6 Ｎ + 行き + の + 片道切符 + を + ～枚 + ください 請給我前往Ｎ的單程票～張
  • 東京行きの片道切符を一枚ください。
  • 請給我一張前往東京的單程票。
  • ※ 「片道切符」為單程票，用於日常對話中，「片道乗車券」的表達方式則較正式。

文法句型

  • 試著用句型造句
# 句型結構 中譯 例句
1 Vて、V〜〜 表達連續或逐一執行的動作
  • ➀、まっすぐ行って、左折して、銀行があります。
  • 直走然後左轉，銀行就在那裡。
  • ➁、地下鉄に乗って、２駅で降りて、左に曲がってください。
  • 搭乘地鐵，兩站下車，然後請左轉。
  • ※ 當一句話中，有多個動詞就會用Ｖて。
2 Vて + から Ｎ之後
  • 切符を買ってから、改札を通ります。
  • 買了票之後，才能通過驗票閘口。
  • ※ 用來表示兩個動作或事件的時間先後順序。
3 N は + 場所 + に + あります 在場所有Ｎ
  • ➀、改札口は駅の入り口にあります。
  • 閘口在車站的入口。
  • ➁、図書館は学校の近くにあります。
  • 圖書館在學校附近。
  • ※ 用來描述某個物品或者事物的位置。
4 N + は + どこですか Ｎ在哪裡？
  • ➀、エレベーターはどこですか。
  • 電梯在哪裡？
  • ➁、駅はどこですか。
  • 車站在哪裡？
  • ※ 詢問某個物品或者事物的位置。
5 これ + は + N + 行きですか 這個是去Ｎ的嗎？( 可以乘坐哪條線去Ｎ地點？)
  • ➀、渋谷でどの線で行けますか。
  • 可以乘坐哪條線去澀谷？
  • ➁、東京駅でどの線で行けますか。
  • 可以搭乘哪條線前往東京站？
  • ※ 來詢問在特定地點可以搭乘的交通線路，確認路線或交通選擇。
6 私は + N + へ + 行きたいです 我想要去～～
  • ➀、私は銀座へ行きたいです。
  • 我想去銀座。
  • ➁、私は新宿へ行きたいです。
  • 我想去新宿。
  • ※ 表達自己想要去某地點。
7 目的 + まで + N + を + 数量詞 + ください 請給我到目的Ｎ數量詞
  • ➀、東京までの切符を一枚ください。
  • 請給我一張到東京的票。
  • ➁、空港までの乗車券を二枚ください。
  • 請給我兩張到機場的乘車券。
  • ➂、美術館までの入場券を三枚ください。
  • 請給我三張到美術館的入場券。
  • ※ 表達自己想要買的票卷抵達哪裡、購買幾張。

搭公車 & 計程車

今天的行程路線看起來適合搭公車，但是要注意末班車的時間，如果趕不上就要叫計程車了。

  • 情境

    搭公車也很方便，但回程可能會趕不上公車末班車，到時候需要叫計程車回家。

  • 任務說明

    找到站牌並詢問能否抵達某地點，而搭計程車時能表達想前往什麼地點、在哪下車。

  • 學習要點

    • ➀、練習詢問公車站牌在哪裡、詢問公車是否有抵達某地點。
    • ➁、練習詢問公車車票費用多少。
    • ➂、搭乘計程車時，詢問車程費用、表達想前往什麼地點，請求司機在哪裡下車。

迷路時問路

在陌生的城市走著走著，不知不覺就迷失了方向，詢問路人或附近的店員來尋找正確的方向。

  • 情境

    雖然看著 Google Map 上的路線走，但好像走錯路了。

  • 任務說明

    試著向路人禮貌地詢問，幫助自己找到方向。

  • 學習要點

    • ➀、能夠詢問自己離想抵達的目的地有多遠。
    • ➁、練習說出自己想去的地點，詢問如何前往。
    • ➂、能聽出「方向」的關鍵字。

  • 作答說明

    下載作答紙，完成「聽寫題目」中挖空的句子，將答案寫進作答紙中。

文法句型

  • 試著用句型造句
# 句型結構 中譯 例句
1 N + から + N + まで + は + 遠いですか 從Ｎ到Ｎ遠嗎？
  • ➀、駅からホテルまでは遠いですか。
  • 從車站到飯店遠嗎？
  • ➁、空港から市中心までは遠いですか。
  • 從機場到市中心遠嗎？
  • ※ 「Ｎから」表示「從...」，「Ｎまで」表示「到...」。
2 N + は + N + の向こうに + あります Ｎ在Ｎ的對面
  • 郵便局は図書館の向こうにあります。
  • 郵局在圖書館的對面。
  • ※ 「Ｎの向こうに」表示在...的對面，描述兩個地點之間的位置關係。
3 N + まで + まっすぐ行って + から + 左に曲がって + ください 請直走到Ｎ，然後左轉
  • 駅までまっすぐ行ってから左に曲がってください。
  • 請直走到車站，然後左轉。
  • ※ 「左に曲がって」為左轉，「右に曲がって」為右轉。
4 Nへの + 一番早い道 + は + どちらですか 往Ｎ最快的路怎麼走？
  • 空港への一番早い道はどちらですか。
  • 去機場最快的路是怎麼走？
  • ※ 「へ」用於表示方向。
5 N + は + N + の + 後ろに + あります Ｎ在Ｎ的後面
  • 銀行はコンビニの後ろにあります。
  • 銀行在便利商店的後面。
  • ※ 「Ｎの後ろに」表示「在 N 的後面」。
6 近くの + N + は + どうやって + 行きますか 附近的Ｎ該怎麼去？
  • 近くの駅はどうやって行きますか。
  • 附近的車站該怎麼去？
  • ※ 「どうやって」表示「怎麼樣、如何，詢問方式或方法」。
