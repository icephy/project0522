文法句型

  • 試著用句型造句
# 句型結構 中譯 例句
1 N + が + あります（か） 有Ｎ（嗎）
  • ➀、お金がありますか？
  • 有錢嗎？
  • ➁、ここにトイレがありますか？
  • 這裡有廁所嗎？
  • ※ 詢問某物是否存在，或是否有某樣東西。
2 場所 + に + N + がありますか？ 在場所有Ｎ嗎
  • ➀、公園にトイレがありますか？
  • 公園裡有廁所嗎？
  • ➁、部屋にスリッパがありますか？
  • 房間裡有拖鞋嗎？
  • ※ 詢問某個地方是否有某樣東西：「在某個地方有...嗎？」。
3 N + から + N + まで 從～～到～～
  • ➀、９時から１２時まで仕事をします。
  • 我從早上 9 點工作到中午 12 點。
  • ➁、4月から6月まで。
  • 從 4 月到 6 月。
  • ※ 意思是「從...到...」。
