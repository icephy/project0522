詢問商品款式

想找那件觀望超久的上衣，需要向店員詢問商品在哪裡，希望還有庫存！

  • 情境

    東西好多逛不完，必買的東西一定要先找出來！

  • 任務說明

    向店員詢問服飾款式、顏色。

  • 學習要點

    • ➀、練習詢問商品類型。
    • ➁、能夠說出想要找的款式、色系。

  • 作答說明

    下載作答紙，完成「聽寫題目」中挖空的句子，將答案寫進作答紙中。

同學可以在閱讀完「任務詳情」後，點擊「認識單字」、「文法句型」來學習本關知識點，吸收後試著利用這些知識點來完成任務內容吧！

文法句型

  • 試著用句型造句
# 句型結構 中譯 例句
1 こちらの + N + は + いかがですか 這個Ｎ怎麼樣？
  • ➀、このバッグはいかがですか。
  • 這個包包怎麼樣？
  • ➁、こちらの時計はいかがですか。
  • 這個手錶怎麼樣？
  • ※ 「いかがですか」表示「怎麼樣、如何」。
2 すみません、この店では + N + の種類はありますか 這家店有Ｎ的種類嗎？
  • ➀、すみません、この店では洋服の種類はありますか。
  • 不好意思，這家店有服裝的種類嗎？
  • ➁、すみません、この店では電子機器の種類はありますか。
  • 不好意思，這家店有電子產品的種類嗎？
  • ※ 詢問某家店是否有某種商品或品類。
3 すみません、N + はどこですか Ｎ在哪裡？
  • ➀、すみません、新しいバッグはどこですか。
  • 不好意思，新款包包在哪裡？
  • ➁、すみません、セール商品はどこですか。
  • 不好意思，促銷商品在哪裡？
  • ➂、すみません、支払いカウンターはどこですか。
  • 不好意思，付款區在哪裡？
  • ➃、すみません、お客様サービスはどこですか。
  • 不好意思，顧客服務台在哪裡？
  • ※ 詢問某個物品、特定商品或地點時使用。
4 すみません、この + N + は + N + がありますか 這個Ｎ有Ｎ的嗎？
  • ➀、すみません、この靴は黒い色がありますか。
  • 不好意思，這雙鞋子有黑色的嗎？
  • ➁、すみません、このシャツは L サイズがありますか。
  • 不好意思，這件襯衫有 L 尺寸的嗎？
  • ※ 尋找特定顏色、尺寸等時使用。

文法句型

  • 試著用句型造句
# 句型結構 中譯 例句
1 N + で支払ってもいいですか 可以用Ｎ支付嗎？
  • ➀、スマホで支払ってもいいですか。
  • 可以用手機支付嗎？
  • ➁、電子マネーで支払ってもいいですか。
  • 可以用電子錢包支付嗎？
  • ※ 助詞「で」表示手段、方法或工具。
2 N + を + もう一つもらってもいいですか 可以再拿一個Ｎ嗎？
  • ➀、紙袋をもう一つもらってもいいですか。
  • 可以再拿一個紙袋嗎？
  • ➁、メニューをもう一つもらってもいいですか。
  • 可以再拿一份菜單嗎？
  • ※ 「もう一つ」表示再一個，「いいですか」詢問是否可以。
3 某個動作/服務 + はできますか 可以「某個動作/服務」嗎？
  • ➀、交換はできますか。
  • 可以換貨嗎？
  • ➁、テイクアウトはできますか。
  • 可以外帶嗎？
  • ➂、予約はできますか。
  • 可以預約嗎？
  • ➃、返金はできますか。
  • 可以退款嗎？
  • ➄、配送はできますか。
  • 可以寄送嗎？
  • ※ 詢問是否可以做某件事、允許的可能性。
4 他の + 顏色/尺寸/商品 + に + 交換できますか 可以換其他「顏色/尺寸/商品」嗎？
  • ➀、他のサイズに交換できますか。
  • 可以換其他尺寸嗎？
  • ➁、他の色に交換できますか。
  • 可以換其他顏色嗎？
  • ➂、他の商品に交換できますか。
  • 可以換其他商品嗎？

➁ 短文聽力

短文聽力共有四題，每一題會播放一段簡短內容，同學可以先檢視問題後，再播放來聽出答案。

01

下列敘述何者正確？
  • ➀、客人詢問有沒有灰色的款式。
  • ➁、店員說裙子有黑、藍、白三種。
  • ➂、客人詢問藍色款有沒有 L 尺寸。
  • ➃、店裡沒有 L 尺寸的褲子。

02

下列敘述何者正確？
  • ➀、本店服飾不方便試穿。
  • ➁、客人想試穿 M 尺寸的背心。
  • ➂、客人詢問有沒有試用品。
  • ➃、店裡有 M 尺寸的襯衫。

03

下列敘述何者錯誤？
  • ➀、皮夾應付的價錢是 8000 円。
  • ➁、商品的目前有折扣活動。
  • ➂、商品的原價是一萬日圓。
  • ➃、三樓可以辦理免稅。

04

下列敘述何者錯誤？
  • ➀、包包的拉鍊壞掉了。
  • ➁、店員詢問他換貨意願。
  • ➂、客人決定換另一個包包。
  • ➃、客人辦理退貨要帶收據。
